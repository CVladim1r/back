Card:

Класс для представления карты. Содержит масть и ранг карты.
Метод can_beat проверяет, может ли данная карта побить другую карту с учетом козырной масти.
Player:

Класс для представления игрока. Содержит идентификатор игрока (sid), его руку (список карт) и вебсокет для связи.
Room:

Класс для представления комнаты игры. Содержит идентификатор комнаты, список игроков, колоду карт, козырную карту, текущего игрока и состояние игры.
Методы для инициализации и создания колоды, уведомления игроков о состоянии игры, начала игры, раздачи карт, выполнения хода, защиты и проверки победного состояния.
Игровая логика:

initialize_deck: Создает колоду карт и, если необходимо, перемешивает её.
start_game: Инициализирует игру, раздает карты игрокам и выбирает атакующего и защищающегося.
deal_cards: Раздает карты игрокам до тех пор, пока у них не будет по 6 карт, или пока не закончится колода.
play_card: Выполняет ход игрока, добавляя карту в активные карты и переключая текущего игрока.
defend_move: Выполняет защитный ход, проверяя возможность побить карту атакующего.
check_win_condition: Проверяет, есть ли победитель.
end_turn: Завершает текущий ход, обновляет состояние игры и раздает карты, если это необходимо.
Функции для управления комнатами и игроками:

create_room: Создает новую комнату игры.
add_player: Добавляет игрока в комнату.
start_game: Запускает игру в указанной комнате.
Обработка сообщений от клиентов:

handle_client_message: Обрабатывает действия, такие как ход игрока, защитный ход, раздача карт и проверка победного состояния.
Уведомление игроков:

notify_players: Отправляет текущее состояние игры всем игрокам в комнате.
